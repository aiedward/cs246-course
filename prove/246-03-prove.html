Spiritual Journal XML Parsing

Overview

Your task is to use Java to parse an XML document for a spiritual insight journal, and instantiate classes that relate to the various tags. The XML document has a root element, journal, that contains multiple entry elements. Each entry has a date, and then potential child elements of 0 or more scriptures, 0 or more topics and content. For example, consider the following:

<journal>
 <entry date="2014-12-16">
   <scripture book="Hebrews" chapter="11" startVerse="1" />
   <scripture book="Alma" chapter="32" startVerse="21" endVerse="43" />
   <topic name="Faith" />
   <content>
     Reading in Hebrews 11:1 and Alma 32:21-43 makes me think about faith.
   </content>
 </entry>
 <entry date="2014-12-17">
   <topic name="Love" />
   <content>
     I need to find a way to develop more charity.
   </content>
 </entry>
</journal>

Instructions

For this assignment, you should create the following classes:

  • Journal
  • Entry
  • Scripture
  • Topic

In addition, please make your Scripture and Topic implement the following Annotation interface:

public interface Annotation {
    String getDisplayText();
}

For relationships, your Journal class should contain a list of entries, and your entries should contain a list of annotation objects. For simplicity, you may store the the date of the entries as a String.

Your program should accept a filename as a command line parameter. It should then parse the XML document and create the objects listed above. Finally, after finishing all of the parsing, it should display each entry as follows:

For parsing the XML document in Java, you should use the DOM approach (i.e., use the classes: javax.xml.parsers.DocumentBuilder and org.w3c.dom.Document).

Content:

Note: It's ok if your whitespace isn't perfect before and after the content.

You are encouraged to create your own files for debugging, however, if it is helpful, you may also see an example spiritual journal that may assist you in your testing and development.
