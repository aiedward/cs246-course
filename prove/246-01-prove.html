Individual Assignment: File I/O

Overview

Create a Java program that counts the number of words per line in a text file.

Please note that by design, this assignment (and future assignments) will leverage some of the fundamentals from the reading/quizzes, but it will also require you to do some digging on the Internet and working with your classmates to find solutions outside of the reading. This skill of learning to find good solutions is critical to your success in this class as well as in developing software in the real world.

Instructions

  1. Create a Java program that accepts a filename as a command line parameter.
  2. The program should open the file read through it line by line.
  3. For each line, display the number of words on that line followed by a colon and then the text of the line. You may assume that spaces separate all words.
  4. If the file cannot be opened, the program should display an error message and exit gracefully.

Example

java WordCounter woodruff.txt

Opening file 'woodruff.txt'...

7: You may have the administration of angels;
5: you may see many miracles;
8: you may see many wonders in the earth;
10: but I claim that the gift of the Holy Ghost
10: is the greatest gift that can be bestowed upon man.
8: -Wilford Woodruff, Deseret Weekly, April 6, 1889, 451.

or like this:

java WordCounter aBadFile.txt

Opening file 'aBadFile.txt'...

Error reading 'aBadFile.txt'...

See the file: 246-01-woodruff.txt as one example, but you should test your code on other files as well.
