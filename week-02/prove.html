Software Design and Development :: CS 246

Prove 2.0 :: Basic Android App

Objectives

  • Learn how to use Intents to communicate between Activities.

  • Learn how to create a second Activity in Android.

Intents

  1. For a score of 93%: Follow this tutorial from Google to add a second activity to your project.

  2. For a score of 100%: Add a cool feature to your Android App, this can be just about anything, you want, but you'll need either two minor embellishments or one major.

    At this stage, major embellishments would include things like extra functionality, displaying the message in a toast, sending additional data through the intent, etc...

    Minor embellishments would include layout and presentation changes.

  3. Be sure to take the corresponding quiz on I-Learn once you are finished.
