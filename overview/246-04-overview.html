Lesson 04 Overview

This week, we have two main objectives: first, formalize our project proposals, and second, learn some tools that we will need for collaborative development.

Objectives

  • Formalize project proposals
  • Become familiar with the basics of a version control system

Individual Preparation

To be prepared to develop software collaboratively, we need to learn about some of the tools that we will be using, In particular, this week we will introduce version control systems. Your individual assignment will walk you through getting set up with the Git version control system and some of the basic principles of it.

Team Project

For your weekly team activity this week, you will work together to solidify the features of your group project. Keep in mind that you can only do so much of this work without feedback and from interaction with your sponsor, so make sure to reach out to them early in the week.

At the end of the week, you will submit a Statement of Work document that outlines your project and the features that you are committing to. Please note that you will need to get your sponsor to sign this document, so you should not wait until the end of the week to put this together.
