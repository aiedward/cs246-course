Lesson Overview

This week we are going to dive into Java a little deeper and use more of the built-in libraries.

The Collections and Interfaces assignment (due Saturday night) will help you learn more about collections in Java and about ways that Java classes can interact, and be packaged.

Don't forget to share ideas and help each other on this assignment. In order to get full credit, you are required to collaborate with at least one other person. The weekly discussion board has been set aside as a place for you to connect with possible collaborators.

In addition, this week, you need to identify potential team members and a basic idea for the project you want to develop this semester. You need to submit your recommendations for team members and project ideas by Saturday night; after that point, the instructor will assign teams.

Lesson Objectives
